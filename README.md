## ğŸš€ Quick Start

### Prerequisites

- Node.js 18+ and npm (recommended: install with npm)
- Web3 wallet (MetaMask recommended)

### Installation

```sh
# Clone the repository
git clone https://github.com/aaravmahajanofficial/forensic-ledger-guardian.git
```
